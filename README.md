# AVE Script

Userscript для блокировки нежелательных продавцов и объявлений на Avito.

## Возможности

- Блокировка продавцов по ID
- Блокировка отдельных объявлений
- Скрытые объявления отображаются в отдельном контейнере
- Автоматическая подгрузка страниц (десктоп)
- Поддержка мобильной версии (m.avito.ru)
- Экспорт/импорт базы данных
- Общая база между десктопом и мобильной версией

## Установка

1. Установите расширение для userscript'ов: [Tampermonkey](https://www.tampermonkey.net/)
2. [Установите скрипт](https://github.com/xob0t/ave-script/releases/latest/download/ave_script.user.js)

## Сборка

```bash
npm install
npm run build
```
